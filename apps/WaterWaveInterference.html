Theory & Formulas

1. Path Difference ($\Delta x$)

The path difference is the difference in distance from the two sources ($S_1, S_2$) to a specific point ($P$).

$$\Delta x = |S_1P - S_2P| = |d_1 - d_2|$$

2. Phase Difference ($\Delta \Phi$)

The total phase difference depends on path difference and initial phase difference ($\phi_0$).

$$\Delta \Phi = \left( \frac{\Delta x}{\lambda} \times 360^\circ \right) + \phi_0 \pmod{360^\circ}$$

3. Conditions for Interference

When sources are in phase ($\phi_0 = 0^\circ$):

  • Constructive: $\Delta x = n\lambda \implies \Delta \Phi = 0^\circ$
  • Destructive: $\Delta x = (n + 0.5)\lambda \implies \Delta \Phi = 180^\circ$
Note: Amplitude attenuation ($1/\sqrt{r}$) is ignored to show mathematical superposition cleanly.
