Physics Principles

This app uses the device's accelerometer to track motion via double integration (Inertial Navigation).

Velocity: $ v(t) = v_0 + \int_{0}^{t} a(t) \, dt $

Displacement: $ s(t) = s_0 + \int_{0}^{t} v(t) \, dt = \iint a(t) \, dt^2 $

Because numeric integration sums errors (noise), the calculated path will drift away from reality over time. Professional systems use Gyroscopes and GPS filters to correct this.
